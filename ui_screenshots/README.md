## 🎨 Screenshot Guidelines

### Quality Standards
- **Format**: PNG (lossless) or JPG (high quality)
- **Resolution**: Full screen resolution (1920x1080 recommended)
- **Color**: True color, no filters
- **Clarity**: Sharp, no blur
- **Lighting**: Good screen brightness

### What to Capture

#### Main Interface
- Full browser window
- All UI elements visible
- Clean, organized layout
- No personal information

#### Detection States
- Clear border colors
- Detection zone visible
- Status text readable
- Confidence scores shown
- Counters updated

#### Feature Demonstrations
- Before and after states
- Button interactions
- Modal dialogs
- Error messages
- Success notifications
